Selecting biologically important variables with PLSKO

#library(PLSKO)

Introduction

This vignette illustrates the basic usage of the package to select important variables in omics data. PLSKO is a method that combines partial least squares (PLS) regression with knockoff filtering to select important variables in high-dimensional biological data with false discovery rate (FDR) control. The package provides functions to generate the knockoff variables, perform the knockoff filtering and aggregate multiple knockoff results.

Here is the outline of the functions corresponding to the workflow of knockoff framework:
